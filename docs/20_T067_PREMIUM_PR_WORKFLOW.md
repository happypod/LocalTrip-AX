# T-067 Premium PR 현장 제작 Workflow

## 목적

Premium PR 제작대행을 현장센터, 앵커조직, 지역 활동가가 실제 운영 상품으로 수행할 수 있도록 표준 절차를 정의한다.

## 1. 운영 단계

1. 문의 접수: `/partner/premium-pr` 신청 접수 후 `/admin/premium-pr-applications`에서 신규 문의 확인
2. 1차 상담: 연락처, 대상 콘텐츠, 제작 가능 항목 확인
3. 제작 범위 확정: 3D/VR 투어, 호스트 인터뷰, 드론 영상, 상세페이지 PR 콘텐츠, 연간 프리미엄 노출 여부 확정
4. 현장 촬영 준비: 촬영 동의, 공간 정리, 출연자 동의, 날씨/소음/동선 확인
5. 콘텐츠 제작: 사진/영상/3D/드론 촬영, 편집, 썸네일 제작, URL 납품 준비
6. 검수: 내부 검수, 사업자 확인, 개인정보/초상권/상표 노출 확인, 오탈자 확인
7. 관리자 등록: `/admin/stays/[id]/edit`에서 Premium PR URL과 메타데이터 입력
8. 공개 확인: `/stays/[slug]`에서 배지/영상 노출과 모바일/PC 확인
9. 운영 리포트 확인: Premium PR 클릭 추적을 월간 운영 보고에 반영
10. 종료/갱신 관리: 노출 종료일 확인, 갱신 상담, 종료 시 Premium PR 해제

## 2. 역할과 책임

| 역할 | 책임 |
|---|---|
| 플랫폼 관리자 | 관리자 페이지 등록, 노출 상태 관리, 리포트 확인 |
| 현장센터 담당자 | 상담, 일정 조율, 제작 범위 확정 |
| 지역 활동가/촬영자 | 현장 촬영, 기본 편집, 업로드 URL 전달 |
| 운영자 | 촬영 동의, 공간 준비, 최종 확인 |
| 검수 담당자 | 개인정보, 초상권, 품질 검수 |

## 3. 촬영 전 체크리스트

- 촬영 동의 확보
- 출연자 동의 확보
- 상호/간판 노출 가능 여부 확인
- 객실/공간 정리
- 화장실, 침구, 주방 등 민감 공간 촬영 범위 확인
- 드론 촬영 가능 여부 확인
- 날씨 확인
- 소음/혼잡 시간대 확인
- 아동/타 방문객 노출 방지
- 차량번호, 연락처, 개인물품 노출 방지

## 4. 검수 체크리스트

- 잘못된 가격/예약확정 문구 없음
- 결제 유도 문구 없음
- 실제 공간과 다른 과장 표현 없음
- 개인정보 노출 없음
- 초상권 동의 없는 인물 노출 없음
- 차량번호, 전화번호, 계좌번호 노출 없음
- 음악/이미지/폰트 저작권 문제 없음
- iframe URL allowlist 충족
- 모바일 숙소 상세 화면 노출 확인
- LeadEvent 수집 확인

## 5. 상태 정의

| 상태 | 의미 | 전환 기준 |
|---|---|---|
| `new` | 신규 접수 | 신청 접수 직후 |
| `in_progress` | 상담/제작 진행 | 담당자가 연락 후 제작 가능성 확인 |
| `resolved` | 등록/공개 완료 | Premium PR URL 등록 및 공개 확인 완료 |
| `archived` | 보관 | 취소, 종료, 중복, 기간 만료 |

## 6. 산출물 기준

- Matterport 또는 3D 투어 URL
- YouTube/Vimeo embed URL
- 대표 썸네일
- 1분 내외 호스트 소개 영상
- 드론 15~30초 클립
- 운영자 확인 완료 메모

실제 파일 업로드 시스템은 구현하지 않고, URL 기반 납품을 기준으로 한다.

## 7. 개인정보/초상권 기준

- 촬영 전 동의를 확보한다.
- 출연자 동의 없는 얼굴 노출은 금지한다.
- 아동 노출은 금지하거나 별도 보호자 동의를 확보한다.
- 차량번호, 전화번호, 계좌번호, 개인 사진/서류 노출을 금지한다.
- 운영자 연락처는 플랫폼 CTA 정책에 맞게 관리한다.
- 삭제 요청 시 운영자가 Premium PR 노출을 해제할 수 있어야 한다.
