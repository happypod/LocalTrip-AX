# MVP Scope: LocalTrip AX / 소원로컬트립

## 포함 범위

- 숙소, 체험, 주민소득상품, 추천코스 공개 노출
- 전화, 카카오, 네이버예약, 홈페이지 CTA 연결
- 문의폼 및 입점신청
- LeadEvent 수집
- 관리자 CRUD
  - Region / BusinessProfile
  - Accommodation
  - Experience
  - LocalIncomeProgram
  - Course
  - Inquiry / PartnerApplication 관리
- 교육·인증·AX 확장을 위한 최소 placeholder 구조

## 제외 범위

- 사용자 회원가입/로그인
- 결제, 정산
- 실시간 예약 확정
- 실시간 객실 재고
- 차량운송
- 후기, 쿠폰, 포인트
- 실제 AI API 연동
- 지도 API 고도화
